How it works

From plug-in to intelligent
home in 15 minutes

No YAML. No cloud accounts. No 40-hour setup. Just plug in, connect to your network, point to your LLM, and start talking to your home.

1

Plug in a PuckNode

Drop it into any outlet. Built-in 3-mic array picks up voice from across the room. USB passthrough so you don't lose the outlet.

2

Install HomeClaw Hub

One command on your Mac, Linux box, or Raspberry Pi. Hub auto-discovers PuckNodes on your network.

3

Connect your model

Point to Ollama running locally, or any API: OpenAI, Anthropic, Groq, Mistral. Swap models anytime.

4

Add your devices

Hue lights, Nest thermostat, Sonos speakers, smart plugs — connect via 80+ built-in integrations or build your own.

5

Talk to your home

Natural language understood by a real LLM, not a keyword matcher. "Run my movie night routine" just works.

The Modes System

One state configures
your entire environment

Every feature connects through HomeClaw's Mode system. Named states that configure lighting, music, climate, security, and notifications — all at once, automatically.

🌅 Morning
trigger: wake time from calendar
  • Sunrise simulation + circadian lighting begins
  • Morning briefing: schedule, weather, tasks
  • Coffee maker starts automatically
🎯 Deep Work
trigger: calendar block or "focus mode"
  • Focus lighting at 5000K, lo-fi music
  • DND on PuckNode LED, surfaces top tasks
  • Notifications silenced, door locked
📹 Meeting
trigger: 5 min before calendar event
  • Video-optimized lighting, music pauses
  • Thermostat holds stable, DND enabled
  • Post-meeting: "Next event at 3pm, 47 min free"
🌙 Evening
trigger: sunset or configurable time
  • Circadian shift: lights warm to 2700K
  • Music shifts to evening playlists
  • Task recap and tomorrow preview
😴 Sleep
trigger: bedtime schedule
  • All lights off, thermostat to 65-68°F
  • Security armed, doors locked
  • 30-min wind-down ramp before activation
🏖️ Weekend
trigger: Saturday-Sunday, no work events
  • Later wake schedule, relaxed lighting
  • Different music profiles loaded
  • Thermostat schedule shifts for leisure
🛫 Away
trigger: last person leaves
  • Thermostat setback, security on
  • Lights off, robot vacuum starts
  • Random light schedule simulates occupancy
👋 Guest
trigger: manual or calendar "guests" event
  • Guest WiFi isolated from HomeClaw control
  • Common areas managed, private spaces excluded
  • Temperature adjusted for extra occupancy
🏋️ Workout
trigger: wearable detects exercise
  • Energizing music, cooler temperature
  • No interruptions during session
  • Hydration reminder after completion
🚨 On-Call
trigger: PagerDuty integration
  • Extended office lighting, alert-ready mode
  • Family DND in other rooms
  • Quiet house except your workspace
🎬 Movie Night
trigger: "movie night" voice command
  • Lights dim to 5%, bias lighting on
  • Music off, thermostat holds stable
  • Full DND, no interruptions
🎉 Party
trigger: "party mode" voice command
  • Color scenes + music-reactive lighting
  • DJ Mode manages the queue all night
  • Thermostat adjusted for higher occupancy
🛑 Emergency
trigger: smoke/CO detector or panic word
  • All lights full white immediately
  • Emergency contacts alerted
  • Doors unlocked for evacuation

Developer First

Real code. Real APIs.

HomeClaw treats developers as first-class citizens. Full CLI, REST + WebSocket API, and automation scripting in JavaScript. Everything is scriptable, everything is subscribable.

terminal 
# Control devices from any terminal
$ homeclaw lights office --brightness 80 --kelvin 4000
✓ Office lights set to 80%, 4000K

$ homeclaw mode deep-work
✓ Deep Work mode activated
  > Lights: 5000K focus profile
  > Music: lo-fi instrumental
  > DND: enabled on all PuckNodes

$ homeclaw status
Mode:     Deep Work
Rooms:    Office (occupied), Kitchen (empty)
Climate:  72°F → target 71°F
Music:    lo-fi beats — Spotify
Next:     Standup in 47 minutes

$ homeclaw webhook list
1. github/pr-merged  → lights.flash(green)
2. github/ci-failed  → lights.flash(red)
3. pagerduty/on-call → mode.set(on-call)
deploy-celebration.js 
// Celebrate successful deploys with your home
import { HomeClaw } from '@homeclaw/sdk'

const hc = new HomeClaw({ host: 'localhost:8420' })

// Listen for GitHub deploy webhooks
hc.on('webhook:github/deploy', async (event) => {
  const { repo, status, author } = event.payload

  if (status === 'success') {
    // Flash office lights green
    await hc.lights.flash('office', {
      color: '#00e5a0', times: 3
    })
    // Announce it
    await hc.say(
      `Deploy to ${repo} succeeded.`
    )
  } else {
    // Flash red — something broke
    await hc.lights.flash('office', {
      color: '#ff5c8a', times: 5
    })
  }
})
